Manual Test for useP2P Null Signaling Fix

This page helps test if the null signaling reference fix is working properly.

Instructions:

  1. Open your browser's developer tools (F12 or right-click and select "Inspect")
  2. Go to the Console tab
  3. Click the "Test P2P with Null Signaling" button above
  4. Watch the console for any errors related to signaling being null
  5. If no errors appear related to "Cannot read property of null" for signaling, the fix is working!
